CSS Text Decoration

In this chapter we will discuss, learn and practice about the following properties:

  • text-decoration-line
  • text-decoration-color
  • text-decoration-style
  • text-decoration-thickness
  • text-decoration

Add a Decoration Line to Text

The text-decoration-line property is used to add a decoration line to text.
We can combine more than one value, like overline and underline to display lines both over and under a text.

Example Code Sample Output
<!DOCTYPE html>
<html>
<head>
     <style>
         #overline { text-decoration: overline; }
         #line-through { text-decoration: line-through; }
         #underline { text-decoration: underline; }
         #overline-underline { text-decoration: overline underline; }
     </style>
</head>
<body>
     <p id="overline"><strong>Overline text decoration</strong></p>
     <p id="line-through"><strong>Line-through text decoration</strong></p>
     <p id="underline"><strong>Underline text decoration</strong></p>
     <p id="overline-underline"><strong>Overline and underline text decoration.</strong></p>
</body>
</html>

Overline text decoration


Line-through text decoration


Underline text decoration


Overline and underline text decoration.

Note: It is not recommended to underline text that is not a link, as this often confuses the reader.

Specify a Color for the Decoration Line

The text-decoration-color property is used to set the color of the decoration line.

Example Code Sample Output
<!DOCTYPE html>
<html>
<head>
     <style>
         #overline-color { text-decoration: overline; text-decoration-color: blue; }
         #line-through-color { text-decoration: line-through; text-decoration-color: red; }
         #underline-color { text-decoration: underline; text-decoration-color: green; }
         #overline-underline-color { text-decoration: overline underline; text-decoration-color: purple; }
     </style>
</head>
<body>
     <p id="overline-color"><strong>Blue: Overline text decoration</strong></p> <br>
     <p id="line-through-color"><strong>Red: Line-through text decoration</strong></p> <br>
     <p id="underline-color"><strong>Green: Underline text decoration</strong></p> <br>
     <p id="overline-underline-color"><strong>Purple: Overline and underline text decoration.</strong></p>
</body>
</html>

Blue: Overline text decoration


Red: Line-through text decoration


Green: Underline text decoration


Purple: Overline and underline text decoration.

Specify a Thickness for the Decoration Line

The text-decoration-thickness property is used to set the thickness of the decoration line.

Example Code Sample Output
<!DOCTYPE html>
<html>
<head>
     <style>
         #overline-thickness { text-decoration: underline; text-decoration-color: green; text-decoration-thickness: auto; /* default */}
         #line-through-thickness { text-decoration: underline; text-decoration-color: green; text-decoration-thickness: 5px; }
         #underline-thickness { text-decoration: underline; text-decoration-color: green; text-decoration-thickness: 20%; }
         #overline-underline-thickness { text-decoration: underline; text-decoration-color: green; text-decoration-style: double; text-decoration-thickness: 2px; }
     </style>
</head>
<body>
         <p id="overline-thickness"><strong>Auto (Default): Underline text decoration</strong></p>
         <p id="line-through-thickness"><strong>5px: Underline text decoration</strong></p>
         <p id="underline-thickness"><strong>20%: Underline text decoration</strong></p>
         <p id="overline-underline-thickness"><strong>2px (with double style): Underline text decoration.</strong></p>
</body>
</html>

Auto (Default): Underline text decoration


5px: Underline text decoration


20%: Underline text decoration


2px (with double style): Underline text decoration.

The Shorthand Property for Text Decoration

The text-decoration property is a shorthand property for:

  • text-decoration-line(required)
  • text-decoration-color(optional)
  • text-decoration-style(optional)
  • text-decoration-thickness(optional)

Example Code Sample Output
<!DOCTYPE html>
<html>
<head>
     <style>
         #tds-one { text-decoration: underline; }
         #tds-two { text-decoration: underline green; }
         #tds-three { text-decoration: underline green double; }
         #tds-four { text-decoration: underline green double 3px; }
         #tds-five { text-decoration: none; }
     </style>
</head>
<body>
     <p id="tds-one"><strong>Text Decoration Shorthand with One Property</strong></p>
     <p id="tds-two"><strong>Text Decoration Shorthand with Two Property</strong></p>
     <p id="tds-three"><strong>Text Decoration Shorthand with Three Property</strong></p>
     <p id="tds-four"><strong>Text Decoration Shorthand with Four Property</strong></p>
     <p id="tds-five">A link with no underline: <a href="https://shahlalhossain.github.io/shahlalhossain">Shahlal Hossain</a></p>
</body>
</html>

All CSS text-decoration Properties

Property Description
text-decorationSets all the text-decoration properties in one declaration
text-decoration-colorSpecifies the color of the text-decoration
text-decoration-lineSpecifies the kind of text decoration to be used (underline, overline, etc.)
text-decoration-styleSpecifies the style of the text decoration (solid, dotted, etc.)
text-decoration-thicknessSpecifies the thickness of the text decoration line
