Text Alignment and Text Direction

In CSS Text Alignment - we will learn about the following properties:

  • text-align
  • text-align-last
  • direction
  • unicode-bidi
  • vertical-align

Text Alignment (Center, Left & Right)

The text-align property is used to set the horizontal alignment of a text.
A text can be left or right aligned, centered, or justified.
The following example shows center aligned, and left and right aligned text.

Example Code Sample Output
<!DOCTYPE html>
<html>
<head>
     <style>
         #heading-center { text-align: center; }
         #heading-left { text-align: left; }
         #heading-right { text-align: right; }
     </style>
</head>
<body>
     <h2 id="heading-center">Heading 1 (Center Aligned)</h2>
     <h2 id="heading-left">Heading 2 (Left Aligned)</h2>
     <h2 id="heading-right">Heading 3 (Right Aligned)</h2>
</body>
</html>

Heading 1 (Center Aligned)


Heading 2 (Left Aligned)


Heading 3 (Right Aligned)

Note: Left Alignment is Default if Text Direction is Left-to-Right and Right Alignment is Default if Text Direction is Right-to-Left.

Text Alignment (Justify)

When the text-align property is set to "justify", each line is stretched so that every line has equal width, and the left and right margins are straight.

Example Code Sample Output
<!DOCTYPE html>
<html>
<head>
     <style>
         #justify-content {
             border: 1px solid black;
             padding: 10px;
             width: 200px;
             height: 200px;
             text-align: justify;
         }
     </style>
</head>
<body>
     <div id="justify-content">
     <p>The text-align: justify; value stretches the lines so that each line has equal width (like in newspapers and magazines).</p>
     <br>
     <p>In my younger and more vulnerable years my father gave me some advice that I've been turning over in my mind ever since. 'Whenever you feel like criticizing anyone,' he told me, 'just remember that all the people in this world haven't had the advantages that you've had.'</p>
     </div>
</body>
</html>

Text Direction

The direction and unicode-bidi properties can be used to change the text direction of an element.

Example Code Sample Output
<!DOCTYPE html>
<html>
<head>
     <style>
         p.left-to-right { direction: ltr; }
         p.right-to-left { direction: rtl; unicode-bidi: bidi-override; }
     </style>
</head>
<body>

<fieldset style="margin: 10px; padding: 20px;">
     <legend style="margin-left: 30px;"><strong>default text direction: left-to-right:</strong></legend>
     <p>This is the default text direction.</p>
</fieldset>

<fieldset style="margin: 10px; padding: 20px;">
     <legend style="margin-left: 30px;"><strong>specified text direction: left-to-right:</strong></legend>
     <p class="left-to-right">This is left-to-right text direction which is specified.</p>
</fieldset>

<fieldset style="margin: 10px; padding: 20px;">
     <legend style="margin-left: 30px;"><strong>specified text direction: right-to-left:</strong></legend>
     <p class="right-to-left">This is right-to-left text direction which is specified.</p>
     <p class="right-to-left">هذا هو اتجاه النص من اليمين إلى اليسار المحدد</p>
</fieldset>

</body>
</html>
default text direction: left-to-right:

This is the default text direction.

specified text direction: left-to-right:

This is left-to-right text direction which is specified.

specified text direction: right-to-left:

This is right-to-left text direction which is specified.

Vertical Alignment

The vertical-align property sets the vertical alignment of an element.

Example Code Sample Output
<!DOCTYPE html>
<html>
<head>
     <style>
         img.a { vertical-align: baseline; }
         img.b { vertical-align: text-top; }
         img.c { vertical-align: text-bottom; }
         img.d { vertical-align: sub; }
         img.e { vertical-align: super; }
     </style>
</head>
<body>

<fieldset style="margin: 10px; padding: 20px;">
     <legend style="margin-left: 20px;"><strong>vertical-align: baseline (default):</strong></legend>
     <p>An <img class="a" src="images/sqpurple.gif" width="10" height="10" alt=""> image with a default alignment.</p>
</fieldset>

<fieldset style="margin: 10px; padding: 20px;">
     <legend style="margin-left: 20px;"><strong>vertical-align: text-top:</strong></legend>
     <p>An <img class="b" src="images/sqpurple.gif" width="10" height="10" alt=""> image with a text-top alignment.</p>
</fieldset>

<fieldset style="margin: 10px; padding: 20px;">
     <legend style="margin-left: 20px;"><strong>vertical-align: text-bottom:</strong></legend>
     <p>An <img class="c" src="images/sqpurple.gif" width="10" height="10" alt=""> image with a text-bottom alignment.</p>
</fieldset>

<fieldset style="margin: 10px; padding: 20px;">
     <legend style="margin-left: 20px;"><strong>vertical-align: sub:</strong></legend>
     <p>An <img class="d" src="images/sqpurple.gif" width="10" height="10" alt=""> image with a sub alignment.</p>
</fieldset>

<fieldset style="margin: 10px; padding: 20px;">
     <legend style="margin-left: 20px;"><strong>vertical-align: sup:</strong></legend>
     <p>An <img class="e" src="images/sqpurple.gif" width="10" height="10" alt=""> image with a super alignment.</p>
</fieldset>

</body>
</html>
vertical-align: baseline (default):

An image with a default alignment.

vertical-align: text-top:

An image with a text-top alignment.

vertical-align: text-bottom:

An image with a text-bottom alignment.

vertical-align: sub:

An image with a sub alignment.

vertical-align: sup:

An image with a super alignment.

Text Orientation

The text-orientation property sets the orientation of the text characters in a line of an element.
It only affects text in vertical mode. It is useful for controlling the display of languages that use vertical script, and also for making vertical table headers.

The CSS Text Alignment/Direction Properties

Property Description
directionSpecifies the text direction/writing direction
text-alignSpecifies the horizontal alignment of text
text-align-lastSpecifies how to align the last line of a text
unicode-bidiUsed together with the direction property to set or return whether the text should be overridden to support multiple languages in the same document
vertical-alignSets the vertical alignment of an element
