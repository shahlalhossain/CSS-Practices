CSS Text Shadow

The text-shadow property adds shadow to text.

Example Code Sample Output
<!DOCTYPE html>
<html>
<head>
     <style>
         p.shadow-normal { text-shadow: 2px 1px; }
         p.color-shadow { text-shadow: 2px 1px green; }
         p.color-shadow-blur { text-shadow: 2px 1px 5px green; }
         p.color-shadow-text { color: white; text-shadow: 2px 1px 4px #000000; }
         p.shadow-red-neon-glow { text-shadow: 0 0 3px #FF0000; }
         p.shadow-red-blue-neon-glow { text-shadow: 0 0 3px #FF0000, 0 0 5px #0000FF; }
         p.shadow-example-one { color: white; text-shadow: 1px 1px 2px black, 0 0 25px blue, 0 0 5px darkblue; }
     </style>
</head>
<body>
<fieldset>
     <legend>Normal Shadow</legend>
     <p class="shadow-normal">Newspapers are the mirror of the world. It plays an important role in modern civilization.</p>
</fieldset>
<fieldset>
     <legend>Shadow with Color</legend>
     <p class="color-shadow">Newspapers are the mirror of the world. It plays an important role in modern civilization.</p>
</fieldset>
<fieldset>
     <legend>Shadow with Color and Blur Effect</legend>
     <p class="color-shadow-blur">Newspapers are the mirror of the world. It plays an important role in modern civilization.</p>
</fieldset>
<fieldset>
     <legend>Shadow with Text Color</legend>
     <p class="color-shadow-text">Newspapers are the mirror of the world. It plays an important role in modern civilization.</p>
</fieldset>
<fieldset>
     <legend>Shadow with Red Neon Glow</legend>
     <p class="shadow-red-neon-glow">Newspapers are the mirror of the world. It plays an important role in modern civilization.</p>
</fieldset>
<fieldset>
     <legend>Shadow with Red and Blue Neon Glow</legend>
     <p class="shadow-red-blue-neon-glow">Newspapers are the mirror of the world. It plays an important role in modern civilization.</p>
</fieldset>
<fieldset>
     <legend>Another Complex Shadow on Text</legend>
     <p class="shadow-example-one">Newspapers are the mirror of the world. It plays an important role in modern civilization.</p>
</fieldset>
</body>
</html>

CSS Box Shadow

The box-shadow property is used to apply one or more shadows to an element.

Example Code Sample Output
<!DOCTYPE html>
<html>
<head>
     <style>
         .box-shadow-normal { border: 1px solid; box-shadow: 10px 10px; }
         .box-shadow-color { border: 1px solid; box-shadow: 10px 10px lightblue; }
         .box-shadow-blur-color { border: 1px solid; box-shadow: 10px 10px 5px lightblue; }
         .box-shadow-multi-color { border: 1px solid; box-shadow: 5px 5px blue, 10px 10px red, 15px 15px green; }
         .box-shadow-multi-color-blur { border: 1px solid; box-shadow: 5px 5px 8px blue, 10px 10px 8px red, 15px 15px 8px green; }
     </style>
</head>
<body>

<div class="box-shadow-normal">
     <h4 style="padding: 10px;">Box Shadow - Horizontal and Vertical</h4>
     <p style="padding: 10px;">Newspapers are the mirror of the world. It plays an important role in modern civilization.</p>
</div>

<div class="box-shadow-color">
     <h4 style="padding: 10px;">Box Shadow with Color</h4>
     <p style="padding: 10px;">Newspapers are the mirror of the world. It plays an important role in modern civilization.</p>
</div>

<div class="box-shadow-blur-color">
     <h4 style="padding: 10px;">Box Shadow - with Blur Color</h4>
     <p style="padding: 10px;">Newspapers are the mirror of the world. It plays an important role in modern civilization.</p>
</div>

<div class="box-shadow-multi-color">
     <h4 style="padding: 10px;">Box Shadow - Multi Colors</h4>
     <p style="padding: 10px;">Newspapers are the mirror of the world. It plays an important role in modern civilization.</p>
</div>

<div class="box-shadow-multi-color-blur">
     <h4 style="padding: 10px;">Box Shadow - Multi Colors with Blur Effect</h4>
     <p style="padding: 10px;">Newspapers are the mirror of the world. It plays an important role in modern civilization.</p>
</div>

</body>
</html>
